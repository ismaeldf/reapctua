# Repactua API

## Requisitos

- Node.js >= 8.0.0
- npm >= 3.0.0
- Adonis >= 4
- Docker and Docker Compose

## Instalação de requisitos

## Node + npm

https://nodejs.org/en/download/

## Docker

https://docs.docker.com/compose/install/

## Adonis

```
  npm i -g @adonisjs/cli
```

## Instalando packages

```
npm install
```

## Executando o docker

```
docker-compose up -d
```

## Executando o migration

```
adonis migration:run
```

## Executando a aplicação

```
adonis serve --dev
```
